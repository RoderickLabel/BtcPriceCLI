# BtcPriceCLI
Com ele é possível a consulta de cotações nas mais diversas moedas, como: EUR, GBP, JPY, CLP, KRW, PLN, USD, AUD, ILS, BRL, CAD, MYR, ZAR, RUB, IDR, NGN, e VND.

## Requirements
  * Python3

## Usage

Inicie o app
```sh
$ python3 app.py
```

Escolha a moeda pelo seu código ISO:
```
>USD
```
